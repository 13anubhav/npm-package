Anubhav npm Package Guide

Welcome to this comprehensive guide on creating and publishing your own npm package. Follow these steps to get started:

  1. Initialize your package: npm init

    Add your code to the index.js file. Here's an example of a simple prime number checker:

    // Function to check if a number is prime
function isPrime(number) {
    if (number <= 1) return false;
    for (let i = 2; i <= Math.sqrt(number); i++) {
        if (number % i === 0) return false;
    }
    return true;
}

// Export the isPrime function
module.exports = isPrime;
  2. Create a local link: npm link

    This creates a local link to install your npm files from your local machine.

  3. Test your package:

    Install your package locally:

    npm i package-name

    Use it in your test file:

    const isPrime = require('check-if-prime');
console.log(isPrime(79));
  4. Publish your package: npm publish

    Make sure you're logged in to npm and have a unique package name before publishing.
